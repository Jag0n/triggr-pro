### 1. 10m Air Pistol — Men

| Parameter | Specification |
|-----------|--------------|
| **Discipline** | Pistol |
| **Distance** | 10 metres |
| **Calibre** | 4.5mm (.177) air pistol |
| **Max gun weight** | 1.5 kg |
| **Position** | Standing, one hand, unsupported |

#### Qualification Format
| Parameter | Detail |
|-----------|--------|
| Total shots (Qual) | 60 competition shots |
| Time limit | 75 minutes |
| Sighters | 10 minutes unlimited sighting shots before qual begins |
| Series structure | 60 shots, typically shot in series of 10 |
| Scoring | Decimal (e.g. 10.2, 9.8) — ISSF electronic targets |
| Integer scoring | Used on paper targets at lower-level competitions |
| Max possible score | 600.0 (decimal) / 600 (integer) |

#### Final Format (Olympic/ISSF World Cup)
| Parameter | Detail |
|-----------|--------|
| Finalists | Top 8 from qualification |
| Qual score carry-forward | No — all finalists start from 0.0 |
| Total final shots | 24 shots |
| Shot tempo | 50 seconds per shot |
| Elimination | After every 2 shots, lowest scorer eliminated (starting after shot 10) |
| Series in final | Single series of 24 shots with progressive elimination |
| Scoring in final | Decimal (10-ring target, inner-10 counts) |
| Medal matches | Gold: last 2 shooters, 5 shots each; Bronze: separate 5-shot match |

#### Target Specifications
| Zone | Diameter (mm) |
|------|--------------|
| 10-ring (inner-10 / X) | 11.5 mm |
| 9-ring | 29.5 mm |
| 8-ring | 51.5 mm |
| 7-ring | 75.5 mm |
| 6-ring | 103.5 mm |
| Overall target size | 170 × 170 mm (paper) / as per EST system (electronic) |

### 3. 25m Rapid Fire Pistol — Men

| Parameter | Specification |
|-----------|--------------|
| **Discipline** | Pistol |
| **Distance** | 25 metres |
| **Calibre** | .22 LR rimfire, semi-automatic |
| **Max gun weight** | 1.4 kg |
| **Position** | Standing, one hand, unsupported |
| **Olympic status** | Yes (Men only) |

#### Qualification Format
| Parameter | Detail |
|-----------|--------|
| Total shots (Qual) | 60 competition shots |
| Series structure | 6 series × 5 shots each per stage; 2 stages (Stage 1: 30 shots @ 8s, 6s, 4s; Stage 2: 30 shots same) |
| Time limits per series | 8 seconds / 6 seconds / 4 seconds (rotating) |
| Sighters | 2 series × 5 shots before each stage (not scored) |
| Scoring | Hit or Miss (integer 0 or 1 per shot); 5-target bank |
| Max possible score | 60 |
| Target bank | 5 silhouette targets appearing simultaneously |

#### Final Format
| Parameter | Detail |
|-----------|--------|
| Finalists | Top 8 |
| Qual carry-forward | No |
| Final structure | 5 series × 5 shots = 25 shots per stage; 2 stages |
| Time limit | Same rotating 8s/6s/4s per series |
| Scoring | Hit (1) or Miss (0); target must fall or register hit within zone |
| Elimination | After each stage, lowest scorer(s) eliminated |
| Hit zone | 9.5 zone on 25m RFP silhouette target |

#### Target Specifications
| Parameter | Detail |
|-----------|--------|
| Target type | Silhouette (rectangular) |
| Scoring zone | Within the black silhouette outline |
| ISSF final hit zone | 9.5 zone of the standard 25m target |

### 8. 10m Air Rifle — Men

| Parameter | Specification |
|-----------|--------------|
| **Discipline** | Rifle |
| **Distance** | 10 metres |
| **Calibre** | 4.5mm (.177) compressed air or CO₂ |
| **Max gun weight** | 5.5 kg |
| **Position** | Standing only |
| **Olympic status** | Yes |

#### Qualification Format
| Parameter | Detail |
|-----------|--------|
| Total shots (Qual) | 60 competition shots |
| Time limit | 75 minutes |
| Sighters | 15 minutes, unlimited shots |
| Series structure | No mandatory series — continuous 60 shots |
| Scoring | Decimal (e.g. 10.5, 9.8) on electronic targets |
| Max possible score | 654.0 (decimal) / 600 (integer) |

#### Final Format
| Parameter | Detail |
|-----------|--------|
| Finalists | Top 8 |
| Qual carry-forward | No |
| Total final shots | 24 shots |
| Shot tempo | 50 seconds per shot |
| Elimination | After each 2 shots post-shot 10, lowest eliminated |
| Scoring | Decimal, inner-10 (X-ring) tracked |
| Gold/Bronze match | Last 2: 5 shots each; Bronze: separate match |

#### Target Specifications
| Zone | Diameter (mm) |
|------|--------------|
| Inner-10 (X) | 0.5 mm |
| 10-ring | 0.5 mm (electronic) / 1.0 mm paper |
| 9-ring | 5.0 mm |
| 8-ring | 15.0 mm |
| 7-ring | 25.0 mm |
| Overall target size | 45.5 × 45.5 mm (paper) |

> **Note:** On electronic scoring targets (EST), the scoring rings are proportionally defined; the inner-10 is the physical pellet impact point scored to one decimal.

## NRAI-SPECIFIC RULES & VARIATIONS

### Key Differences from ISSF Rules

| Rule Area | ISSF Standard | NRAI Variation |
|-----------|--------------|----------------|
| Scoring system | Decimal (EST mandatory at World/Olympic level) | Integer allowed at State and below; EST mandatory at NSCC |
| Final format | Mandatory at Olympic/WC/World Cup events | Finals conducted at NSCC level; optional at AIGVMSC |
| Qual carry-forward | Not carried to final | Same — no carry-forward at NSCC |
| .22 OS events | Not recognised | Full national event with MQS system |
| Age categories | Senior / Junior / Youth | Senior / Junior / Youth / Sub-Youth / Masters / Senior Masters / Super Masters |
| MQS system | No MQS for ISSF | Strict MQS per event per category for AIGVMSC and NSCC entry |
| Entry eligibility | National federation discretion | MQS must be achieved at State Championship or AIGVMSC |
| Sighting shots | As per technical rules | Same as ISSF; monitored by Range Officer |

### Age Categories (NRAI)

| Category | Age Range |
|----------|-----------|
| Sub-Youth | Under 14 |
| Youth | 14–17 |
| Junior | Under 21 |
| Senior | 21 and above |
| Masters | 45–59 |
| Senior Masters | 60–69 |
| Super Masters | 70+ |

### Competition Tier Hierarchy

| Tier | Competition | Scoring Standard |
|------|-------------|-----------------|
| 1 (Highest) | NSCC — National Shooting Championship | ISSF rules + EST mandatory |
| 2 | AIGVMSC — All India G V Mavlankar | ISSF rules; EST where available |
| 3 | Zonal Championships | NRAI rules; paper/EST |
| 4 | State Championships | NRAI rules; paper targets acceptable |
| 5 | Pre-State / Club | NRAI rules; paper targets |
